# SEO Weekly Process

Use this process every week to continuously improve rankings and CTR.

## Inputs

- Google Search Console: Search results + Pages report
- Google Search Console: Indexing/Coverage report
- Production sitemap health (`/sitemap/0.xml` and chunks)
- Current scorecard file: `docs/seo-weekly-scorecard.csv`

## Weekly routine (45-60 minutes)

1. Pull week-over-week deltas:
   - indexed pages
   - non-brand impressions
   - non-brand clicks
   - average CTR
   - average position
2. Record values in `docs/seo-weekly-scorecard.csv`.
3. Identify:
   - top gain query
   - top loss query
4. Make 1-3 focused SEO changes:
   - metadata title/description updates
   - internal link improvements
   - city/category page intro quality improvements
5. Capture next-week actions and owners.

## Monthly review

- Evaluate which city/category templates drive the most non-brand clicks.
- Expand winning patterns to new city/category combinations.
- Prune or noindex low-value thin pages if needed.
